# QuickPlay

Minimalistinen, nopea äänisoitin — **tuplaklikkaus → soi heti**. Ei kirjastoa,
ei skannausta. Showtime-tyylinen kehyksetön pyöreä ikkuna, kansikuva täyttää,
kontrollit sulautettu alas.

- Close ✕ / minimize – oikeassa yläkulmassa (ei title-baria)
- Vedä mistä tahansa siirtääksesi ikkunaa
- Space = play/pause, ←/→ = kelaa 5s, Esc = sulje
- Monta tiedostoa → jono (prev/next)

Vaatii: python + PySide6 (Qt6 Multimedia, ffmpeg-backend).
Oletussoitin äänelle: `quickplay.desktop`.
